FiveM Development

FiveM is a modded client running on Grand Theft Auto 5’s Engine and assets. Around July of 2020, I began to work on vehicle modifications, such as physics/handling tuning, texture optimization, model editing, and some light scripting. The scripts were to help me measure vehicle statistics for testing. I have been doing this for a few semi-popular Roleplay servers since I began in 2020. It is not a formal place of employment, though I think it’s good to mention the things I've learned while working on a small team that manages a FiveM server.

Link to the Dev Diary of the modification.

Responsibilites

Vehicle Metadata

Organize XML type Vehicle metadata so that it’s usable and functional.

Vehicle Textures

Optimize vehicle textures to minimize virtual memory requirements, while maintaining the visual quality for aesthetics and appearance.

Vehicle Handling

Improve vehicle handling to maintain a “meta” or “balancing” with set vehicle classes having their own range of acceleration, handling and top speed statistics for over 200 cars.

Vehicle Meshes

Modify vehicle meshes in 3D software to patch holes, create UV maps, fix material issues, and create additional features like additional usable seats. + Other minor fixes.

Teamwork

Work closely with a server team, listening to player feedback and working out fast and efficient ways to deliver information and new fixes and improvements.

Quality of Life fixes

Develop additional features to improve quality of life for the players.
